# Ошибка `You must be creator, or administrator`

В случае, если пользователь подключает каналу к Телепосту через бота [{{ telepost.bot.name }}]({{ telepost.bot.url }}), и после этот пользователь выходит из канала или перестает быть в нем админом / владельцем, то при попытке подлкючить такой канал — в веб-версии появится ошибка `You must be creator, or administrator`:

![image](/images/254659536-b2284950-c780-4e00-89ff-7e07fa73f8b4.png)

## Решение

Пользователю, подключившему канал через бота [{{ telepost.bot.name }}]({{ telepost.bot.url }}) — нужно снова стать админом / владельцем в канале.

Также рекомендуется использовать свой кастомный бот, как его подключить — читайте в [этой инструкции](../start/personal-bot-for-telepost.mdx).
